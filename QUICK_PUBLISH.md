# 立即发给别人

如果你现在就要发给别人，最省事的方式是：

## 方式一：GitHub Pages

1. 新建一个 GitHub 仓库
2. 上传这些文件到仓库根目录：
   - `index.html`
   - `styles.css`
   - `app.js`
   - `favicon.svg`
   - `.nojekyll`
   - `vendor/ffmpeg/`
3. 打开仓库 `Settings -> Pages`
4. 选择：
   - `Deploy from a branch`
   - `main`
   - `/ (root)`
5. 保存，等几分钟
6. GitHub 会给你一个公开链接，直接发别人

## 方式二：直接用我整理好的发布包

我已经给你准备一个发布包，你把里面文件解压后按上面的方式上传即可。

## 注意

- 这是纯前端工具，不需要后端
- 10 人内试用通常不用花钱
- 最稳的用法是“本地上传视频”
- 视频越短、镜头越清晰，结果越稳
