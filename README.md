# Seedance 视频拆段提示词分享版

这是一个纯前端静态分享版，适合直接发布到 `GitHub Pages` 或 `Cloudflare Pages`。

如果要继续升级成“前端上传 + 后端异步分析”的版本，后端骨架已经放在：

- `backend/README.md`
- `backend/app/main.py`

当前版本已经支持：

- 上传 2 分钟内视频
- 或粘贴可直接访问的视频直链
- 按 `15s` 自动拆段
- 使用 `FFmpeg.wasm` 在浏览器端真实导出 MP4 切片
- 使用 `Tesseract.js` 识别关键帧中的画面文字
- 为每一段生成可编辑的 `Seedance 2.0` 视频提示词
- 导出 `TXT` 和 `JSON`
- 适合直接静态托管分享给 10 人内小范围使用

## 直接使用

建议用静态服务打开，而不是双击本地文件。

如果你要本地稳定测试，建议在当前目录启动一个静态服务，例如：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/
```

## 当前能力

- 纯浏览器运行，无后端
- 支持本地视频上传
- 支持视频直链导入，但会受到跨域限制
- 自动生成每段缩略图
- 每段都可以手动编辑画面描述、动作描述、OCR 文字和镜头建议
- 可复制单段 Prompt
- 可导出单段 MP4
- 可批量导出全部切片
- 可预加载 FFmpeg 和 OCR 资源

## 使用提醒

- FFmpeg 首次加载大约 30MB，首次打开会稍慢
- OCR 首次加载需要下载语言模型
- “智能”部分目前仍以规则模板 + OCR 辅助为主
- 非直链平台链接通常无法直接处理
- 浏览器性能较弱时，批量导出会比较慢
